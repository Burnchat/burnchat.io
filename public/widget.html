FAQ

  
Is BurnChat widget free?
Yes, completely free. No signup, no credit card, no usage limits. Embed it on as many pages as you want.

  
Does BurnChat store messages?
No. All messages exist only in server memory (RAM). When the room empties or gets burned, everything is destroyed permanently. There is no database.

  
Can I customize the widget?
Yes. You can customize theme (dark/light/auto), accent color, position, size, welcome message, and embed mode (floating bubble or inline).

  
Does it work on WordPress, Shopify, Wix?
Yes. The widget is a single script tag that works on any website — WordPress, Shopify, Wix, Squarespace, Webflow, React, Vue, Next.js, and any custom HTML site.

  
Will the widget slow down my website?
No. The loader is under 5KB, loads async, and the chat iframe only loads when a visitor clicks the chat button. Zero impact on your page speed scores.

  
Can I remove the "Powered by BurnChat" branding?
We're working on a premium tier that removes branding. For now, all widgets include a small "Powered by BurnChat" link.
